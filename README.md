# Hierarchical_zBuffer
## Environment
```
Ubuntu 18.04
CMake 3.18.4
C++ 7.5.0
freeglut 2.8.1 (for display)
```
## Complile project
```bash
cd $ROOT
mkdir build
cmake ..
make
```
## Render
```bash
cd build
./main $PATH_TO_OBJECT
```
